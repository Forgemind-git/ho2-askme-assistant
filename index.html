The pattern

How every sample works

Each is a single-file artifact that hardcodes your real material and calls window.claude.complete() to answer live — no server, no key.

Make one yours — two ways

In Claude.ai, turn on Settings → Feature preview → "Create AI-powered artifacts" (one time).

Prompt Claude: paste a sample's example prompt into a new chat with your details — Claude rebuilds the page as an Artifact you refine in the chat.

Or edit the file: open a sample's index.html, find the EDIT HERE banner in the script, and replace the PROFILE text (and the visible About / cards) with your own.

Click Publish to get a shareable link. Paste that link into the sample README where it says [paste your published artifact share link here].

Anyone with a Claude account can now open the link and ask your page questions live.

Good to know: Anthropic changed how window.claude.complete behaves on 2025-07-31. If live answers stop working, open your artifact inside claude.ai and confirm the current way to call Claude — the feature may have shifted since these examples were written. No API key is ever required for the course.
Use your Claude.ai Pro plan wisely
  • Use the example prompt in each sample's README — it's already written and tested. Don't reinvent it.
  • One clear prompt beats lots of vague back-and-forth.
  • Start a new chat when you switch tasks — long chats use your limit faster.
  • Don't paste big files over and over. Paste once, then refer back to it.
