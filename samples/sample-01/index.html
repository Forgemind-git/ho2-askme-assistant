Build an AI-powered “Ask-Me” assistant

A chat that answers questions from your own info — published as a shareable Claude Artifact. No code and no API key: you build it by describing it in a Claude chat, and it runs live on your Claude subscription.

What you’ll build

An “Ask-Me” assistant: a small chat, grounded only in the facts you give it, that answers as you in the first person and never invents anything you didn’t tell it. Anyone with a Claude account can open the shared link and put real questions to it.

Follow the steps below to build your own. Scroll to the bottom to see a finished example — Maya Chen’s recruiter-facing assistant — running live inside claude.ai.

Build it in Claude chat — step by step

  1. Turn on AI-powered artifacts

    In claude.ai, open Settings → Feature preview and switch on “Create AI-powered artifacts.” This lets an artifact call Claude while it runs — that’s what makes your assistant answer live.

  2. Start a new chat and paste this build prompt

    Open a new chat and paste the prompt below. It tells Claude exactly what to build and how it must behave.

    Build an AI-powered artifact: an "Ask-Me" chat assistant for my job search.

It should speak in the first person as me, a frontend developer, and answer a
recruiter's questions using ONLY the FACTS below. Rules:
- Warm, specific and concise — 2 to 4 sentences.
- Never invent employers, dates, skills or results that aren't in the FACTS.
- If something isn't in the FACTS, say you don't have that detail instead of guessing.

Show it as a clean chat with 3 suggested starter questions and an input box.

FACTS:
<paste your info — see the next step>

  3. Swap in your own info

    Replace the FACTS block with your real details — this is the only material your assistant may use, so keep it short and true.

    Name: <your name>
Target role: <e.g. Frontend Developer, mid to senior>
Experience: <years + where you've worked, one line>
Skills: <the tools and areas you actually know>
Leadership: <any mentoring / leading — or delete this line>
Project 1: <what you built> — Result: <the number or outcome>
Project 2: <what you built> — Result: <the number or outcome>
Looking for: <the kind of team / role you want>

  4. Test it live

    Claude renders a live chat Artifact on the right. Sign in if asked, then try three real questions:

    Have you led a team?
Do you know React and TypeScript?
What's a project you're proud of?

    Every answer should stay true to your facts. If one drifts, just reply in the chat: Only use the FACTS — don't invent anything and Claude rebuilds it.

  5. Publish and share

    Open the artifact’s Share menu, publish it, and copy the link. Note: it only runs inside claude.ai, and each visitor needs their own Claude account to ask it live — it won’t answer if you host the plain HTML on a normal website.
