Write a SQL query to find top 10 customers by LTV
ASSISTANT
Here's a PostgreSQL example: SELECT customer_id, SUM(order_total) AS lifetime_value, COUNT(order_id) AS total_orders, MIN(created_at) AS first_order, MAX(created_at) AS last_order FROM orders WHERE status = 'completed' GROUP BY customer_id ORDER BY lifetime_value DESC LIMIT 10; Tip: Join to a customers table to pull in name/email for reporting.

What does a sudden drop in conversion rate mean?
ASSISTANT
Work through this structured checklist: 1. Data quality — Was the tracking tag firing correctly? Check for GA4 / pixel gaps on the exact dates. 2. Traffic mix — Did a new paid campaign bring in lower-intent visitors? Compare CVR by channel. 3. Product/UX change — Did anything ship to the checkout or landing page? Cohort split — Is the drop uniform or isolated to mobile/new users/one geo? Narrow it to one root cause before acting.
